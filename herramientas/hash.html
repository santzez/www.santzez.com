Generador de hash SHA-256

Herramienta interna para generar el hash de una contraseña antes de añadirla al archivo web/js/usuarios.json. La contraseña no se envía a ningún servidor: el cálculo ocurre íntegramente en tu navegador.

—
—
Cómo usarlo:
  1. Escribe el usuario y la contraseña deseados.
  2. Copia la línea JSON de abajo.
  3. Pégala dentro del array "usuarios" del archivo web/js/usuarios.json.
  4. Sube el archivo actualizado al servidor.
Formato esperado del archivo: 
{
  "usuarios": [
    { "usuario": "lu", "hash": "..." },
    { "usuario": "otro", "hash": "..." }
  ]
}
